    # SaMD（プログラム医療機器）該当性の自己確認報告書

> 対象: `watchlist` (B2Pharmacy)  
> 確認日: 2026-03-05  
> 確認者: AIアシスタント (Antigravity)  

---

## 1. 目的

本ドキュメントは、厚生労働省およびPMDAの「プログラムの医療機器該当性に関するガイドライン」等に基づき、本アプリ（watchlist / B2Pharmacy）が医療機器プログラム（SaMD）に該当するかどうかの自己確認結果を記録するものである。

## 2. アプリケーションの機能概要

本アプリは、以下の機能を主目的としている：
- 厚生労働省が公表する「医薬品の供給状況」データの監視
- 登録された採用薬の供給ステータス変化（通常出荷・限定出荷・供給停止）の自動検知
- ステータス変化があった際、関係者（薬剤師・事務スタッフ等）へのLINE/メール等による通知
- 採用薬全体の供給リスクのダッシュボード表示

## 3. 医療機器該当性の判定

ガイドラインに基づく以下の判定基準に照らし合わせた結果：

### 3-1. 疾病の診断・治療・予防を目的としているか？
- **判定:** 【非該当】
- **理由:** 本アプリは、医薬品の「供給状況（在庫・物流上のステータス）」を扱うものであり、患者個人の疾患に対する「診断」「治療法の選択」「予防」を直接の目的としていない。

### 3-2. 医師等の意思決定を臨床的に支援するものであるか？
- **判定:** 【非該当】
- **理由:** 本アプリが提供する情報は、物流的な在庫確保や代替品の検討（事務的・物流的判断）を支援するものであり、特定の患者に対する処方内容の決定（臨床的判断）を自動で行うものではない。

### 3-3. 医療機器としての副作用、機能不全等が生じた場合に人の生命及び健康に影響を与えるおそれがあるか？
- **判定:** 【極めて低い】
- **理由:** 本アプリの情報に誤りがあった場合でも、それは物流上の支障（発注の遅れ、在庫切れ）に留まる。医療事故に直結するようなバイタルデータの解析や、医療機器の制御は行っていない。

## 4. 判定結果

**本アプリは「在庫管理・発注支援・物流情報参照システム」に類するものであり、医療機器には【非該当】であると判断する。**

ただし、将来的に以下のような機能を実装する場合は、再度判定が必要となる：
- 特定の患者情報（体重・年齢）と紐づけて、具体的な代替薬の「用量」や「処方候補」を自動提示する機能
- 臨床診断基準に基づいたアラート機能

## 5. 運用上の対策

誤解を避けるため、以下の対応を完了している：
- 利用規約において「本サービスは医療機器ではなく、最終判断は専門家の責任において行うこと」を明記。
- 画面上に「データソース（厚生労働省）」を明示し、情報の正確性を保証するものではない旨を表示。
