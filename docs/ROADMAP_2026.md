# Kusuri Compass 2026年 機能拡張ロードマップ

Kusuri Compass（くすりコンパス）の更なる発展に向け、以下の3つの指針に基づいた機能拡張を計画します。

---

## 1. 開発の基本方針
*   **業務支援に特化**: 薬剤師・調剤事務が日々直面する「確認の手間」をテクノロジーで削減する。
*   **個人情報の不保持**: 患者個人を特定する情報は一切取得・保持せず（ステートレス）、安全に利用できるツールを目指す。
*   **法規制の遵守**: 薬機法、医療情報システムに関するガイドライン、公的データ（PMDA/厚労省）に基づいたエビデンスを重視する。

---

## 2. 計画している拡張機能

### 2.1. 薬剤師業務支援（Clinical Excellence）
薬剤師の臨床判断をサポートし、対人業務の時間を創出します。
*   **統合計算ツール**: 
    - 体重別・年齢別の小児力価計算（計算結果のみ表示、保存なし）。
    - 腎機能（eGFR/Ccr）に応じた用法用量確認ガイド。
*   **高度DI（医薬品情報）検索**:
    - 粉砕・経管投与・簡易懸濁の可否情報の一覧化（メーカーインタビューフォーム等から集約）。
    - 配合変化（一包化・混合）時の物理的・化学的安定性データの参照。

### 2.2. 調剤事務業務支援（Admin Efficiency）
処方箋入力や受付での「調べ物」を効率化します。
*   **算定要件ナビゲーター**:
    - 地域支援体制加算、特定薬剤管理指導加算2など、複雑な加算の算定要件をチェックリスト形式で確認。

### 2.4. 実装戦略： Lab（実験用メニュー）の導入
トップメニューの肥大化を防ぎ、安全に検証を行うための仕組みです。
*   **Lab（試験機能エリア）**: 新機能はまず `/lab/` 配下にて実装し、試験運用を行う。
*   **段階的統合**: 現場でのフィードバックを経て、真に必要な機能のみをトップメニューへ昇格させる。

### 2.3. 安全性・ガバナンス（Safety & Governance）
リスクマネジメントと公的情報の迅速な把握を強化します。
*   **PMDA 安全情報・回収情報アラート**:
    - PMDAの公開情報と連携し、緊急安全性情報や自主回収情報をトップページで速報表示。
*   **エビデンス明示の徹底**:
    - AIによる提案（代替薬ナビゲーター等）に対し、常に「根拠（添付文書/ガイドライン）」へのリンクを併記する仕組みの強化。

---

## 3. 実装の優先順位（フェーズ案）

| フェーズ | 重点項目 | 具体的な対象 |
| :--- | :--- | :--- |
| **Phase 1** | 公的データの活用・統合 | PMDA回収情報表示、HOTコード検索の改善 |
| **Phase 2** | 入力負荷の軽減 | 各種計算ツール（小児・腎機能）の単体モジュール追加 |
| **Phase 3** | 高度なナレッジ共有 | 配合変化・粉砕可否データベースの構築 |

---

## 4. 検討事項
- **データ更新の自動化**: 手動メンテナンスを減らすため、GAS等を用いた定期的なスクレイピング・API連携の拡充。
- **モバイル対応**: 薬局内のタブレットやスマートフォンでもストレスなく計算・確認ができるUI/UXの最適化。
